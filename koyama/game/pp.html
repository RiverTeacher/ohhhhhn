プライバシーポリシー

本プライバシーポリシーは、個人が運営するブラウザゲーム「進撃の小山」（以下「本ゲーム」）における、 利用者の情報の取扱いについて定めるものです。

1. 取得する情報

本ゲームでは、利用者個人を特定できる情報（氏名、メールアドレス、LINE ID 等）は一切取得しません。

ただし、ゲームの継続性を確保する目的で、以下の情報を利用者のブラウザ内（Cookie）に保存します。

  • ゲームの進行状況

これらの情報は利用者の端末内にのみ保存され、運営者のサーバーへ送信・保存されることはありません。

2. Cookieの利用について

本ゲームでは、ゲームの進行状況を保存する目的に限りCookieを使用しています。

  • Cookieに保存される情報から、利用者個人を特定することはできません
  • Cookieはブラウザの設定により、利用者自身で削除または無効化することが可能です

Cookieを無効化した場合、ゲームの進行状況が保持されない場合があります。

3. アクセス解析・広告について

本ゲームでは、アクセス解析ツールの利用および広告配信は行っていません。

そのため、運営者が利用者のプレイ状況や利用回数を把握することはできません。

4. LINE Front-end Framework（LIFF）の利用について

本ゲームは、LINE株式会社が提供する LINE Front-end Framework（LIFF） を利用しています。

  • 本ゲームの利用にはLINEアカウントが必要です
  • LINEアカウントの認証および情報管理はLINE社が行います
  • 運営者は、利用者のLINE ID、表示名、プロフィール画像等の情報を取得・保存・利用しません

LINEにおける個人情報の取扱いについては、LINE社のプライバシーポリシーをご確認ください。

5. 未成年の利用について

本ゲームは年齢制限を設けていません。

未成年の方が利用する場合は、保護者の理解と同意のもとでご利用ください。

6. プライバシーポリシーの変更

本プライバシーポリシーは、法令の変更や本ゲームの仕様変更に応じて、予告なく改定される場合があります。

改定後のプライバシーポリシーは、本ゲーム上に掲載した時点で効力を生じるものとします。
